## PBS (Process Breakdown Structure)

이 실습에서 완성하려고 하는 게임의 목표는 아래와 같습니다.

* Ship
  * 이동 (좌, 우)
  * Missile 발생
  * 충돌 검사
  * 폭파
* Missile
    * 이동 (up, 지속적인 전진)
    * 충돌 검사
    * 폭파
* Asteroid
    * 이동 (down, 지속적인 전진)
    * 충돌 검사
    * 폭파
